# 🗺️ اختيار الموقع 📂

### (Installation Location)

- اختيار الموقع القيمة الافتراضية هنا هي
- C:\Program Files\nodejs.
- يمكنك تحديد المجلد الذي ترغب في تثبيت Node.js فيه.
- اختيار المكان المناسب يسهل الوصول إلى الملفات لاحقًا.
- يتضمن ذلك نسخ الملفات اللازمة إلى المواقع الصحيحة في النظام وتسجيل البرنامج لكي يمكنك استخدام Node.js على نظامك.

# 📦 خيارات التثبيت

### ⏳ Node.js Runtime

- تثبيت البيئة الأساسية لتشغيل تطبيقات Node.js. ضروري لتشغيل الكود الخاص بك.

### Corepack Manager 🔄:

- يسمح لك بإدارة حزم التطبيقات بطريقة مرنة. يساعد في التعامل مع إصدارات متعددة من مديري الحزم مثل npm وYarn.

### 📦 npm (Node Package Manager)

- يتضمن تثبيت npm، مدير الحزم الرسمي لـ Node.js. يمكنك استخدامه لتثبيت مكتبات وأدوات خارجية بسهولة.

### Online Documentation Shortcuts 📚:

- إضافة اختصارات للوثائق عبر الإنترنت لتسهيل الوصول إلى المعلومات والمساعدة عند الحاجة.

### Add to PATH 🛤️:

- يضيف Node.js إلى مسار النظام (PATH)، مما يسهل تشغيل الأوامر في سطر الأوامر بدون الحاجة إلى تحديد المسار الكامل.

- تأكد من تحديد هذا الخيار إذا كنت تخطط لاستخدام وحدات نمطية تتطلب تجميعًا.
